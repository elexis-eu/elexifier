1. Upload an xml dictionary and create a transformation

Start by clicking NEW DICTIONARY in the bottom left corner and then upload an XML dictionary, define its metadata and the initial transformation rules.

2. Define the transformation specification

Click Edit in the top row to define the individual core elements of the Elexis Data Model. Select the elements in your xml file from the list or define your own Xpath expression.

3. Download your file

1. Upload your PDF and send it to Lexonomy for annotation.
2. Click the link to annotate your dictionary in Lexonomy. Tip: For best results, annotate at least 4 full pages of dictionary entries.
3. Start the machine learning process. The process runs in the background, so you can close the app and come back later.
4. The results of the ML processing can be previewed in Lexonomy.
5. Due to unknown encodings, PDF conversion can result in corrupted characters. If you see corrupted characters in your dictionary, you can map them to proper characters and Elexifier will replace them in the final file.
6. Download your transformed dictionary.
